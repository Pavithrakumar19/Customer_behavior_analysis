# Customer_behavior_analysis

An end-to-end data analytics project that analyzes 3,900 customer transactions to extract actionable business insights on spending patterns, customer segmentation, and product performance. This project demonstrates proficiency in the complete data analytics pipeline: data cleaning and preparation in Python, complex business analysis using SQL, and interactive visualization through Power BI dashboards.
The analysis identifies key revenue drivers, customer segments, and product trends that support data-driven decision-making for retail optimization, marketing strategy, and inventory management.
